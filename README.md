QEXUR: Autonomous Multi-Agent Security Engine

Qexur is a next-generation, AI-driven cybersecurity platform designed 
for real-time vulnerability assessment and autonomous red-teaming.
Built for the modern web, it leverages a Mixture of Agents (MoA) 
architecture to detect, exploit, and provide fix-reasoning for complex security vectors.


Key Features

Super Destroyer Module: A proprietary multi-agent framework that
orchestrates 100+ parallel AI agents for deep-scan penetration testing.

Live Attack Simulator: DNS-verified environment for safe,
real-time stress testing of web infrastructure.

Surgical Fix-Reasoning: Not just detecting bugs,
but providing production-ready code patches using advanced LLM logic.

Agentic Infrastructure: Powered by a customized n8n
orchestration layer and high-inference models (MiniMax/Gemini).


Tech Stack

Frontend: Next.js 15 (App Router), Tailwind CSS, Framer Motion
Backend & DB: Supabase (Auth/Postgres), Azure Functions
AI Orchestration: n8n, Multi-Agent Logic (MoA)
Deployment: Vercel (Edge Functions)


Installation & Setup
Clone the repository: git clone https://github.com/ujjalrajkonwar/qexur-waitlist.git

Install dependencies: npm install

Set up Environment Variables:
Create a .env.local file with your Supabase and AI API keys.

Run Development Server: npm run dev
